# 项目概述 java555

本项目是一套基于SpringBoot的角色权限后台管理系统脚手架，旨在为开发者提供快速开发的实用工具和参考。

## 技术栈

- **后端：** Spring Boot、Spring Security、MyBatis
- **数据库：** MySQL
- **前端：** Vue.js、Element UI

## 功能模块

- 用户管理
  - 用户注册、登录、信息修改
- 角色管理
  - 角色创建、权限分配
- 资源管理
  - 资源访问权限控制
- 权限控制
  - 精细化权限设置

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- **操作系统：** Windows、Linux或MacOS
- **JDK版本：** JDK 1.8 或更高版本
- **数据库：** MySQL 5.7 或更高版本

## 部署步骤

1. 环境准备：确保JDK、MySQL等运行环境正确安装
2. 数据库配置：根据提供的sql脚本初始化数据库
3. 后端服务启动：运行Spring Boot应用程序
4. 前端项目构建与部署
